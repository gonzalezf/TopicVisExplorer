# API reference

Autogenerated from the library docstrings. See the sub-pages for each
module:

  topic-term / doc-topic distributions.
- [`operations`](operations.md) — split, merge, add/remove word,
  exclude document.
- [`models`](models.md) — the `TopicModelAdapter` protocol and bundled

- [`similarity`](similarity.md) — topic-topic distance matrices and
